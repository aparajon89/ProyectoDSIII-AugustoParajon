# ProyectoDSIII-AugustoParajon
Proyecto de Data Science aplicado a retail e-commerce que combina EDA, Deep Learning y NLP para anticipar demanda y gestionar el riesgo de quiebre de stock. El enfoque prioriza interpretabilidad y uso operativo, integrando predicción y lógica de negocio para apoyar decisiones de planificación y abastecimiento.
